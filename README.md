## Scholar Internship Projects
This repository contains 3 beginner-friendly Machine Learning Engineering projects focused on model deployment, data validation, and monitoring. These projects demonstrate practical ML workflows used in real-world production systems.

📌**Project 1 - Package a Model Inference Service**

Built a Machine Learning inference API using FastAPI and Scikit-learn to serve real-time predictions. The model is trained, saved using Joblib, and exposed through REST API endpoints.

**Features**
- Model training
- Model serialization using Joblib
- REST API using FastAPI
- Real-time prediction endpoint
- Swagger UI testing

📌 **Project 2 - Build a Feature Validation Pipeline**

Built a Machine Learning based Depression Prediction System using Scikit-learn and FastAPI with a Feature Validation Pipeline for validating user input data before prediction. Integrated Evidently AI for data drift detection and monitoring to improve model reliability and production readiness.

### Features

* Depression prediction using Machine Learning
* Feature validation pipeline
* Missing value and invalid data checking
* Real-time prediction API using FastAPI
* Model serialization using Joblib
* Data drift detection using Evidently AI
