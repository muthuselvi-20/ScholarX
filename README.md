# Package a Model Inference Service 🚀

**Project Overview**

This project is a Machine Learning inference API built using FastAPI and Scikit-learn.  
The model predicts student scores based on study hours and sleep hours.

---

## Features

- Train ML model
- Save model using Joblib
- FastAPI REST API
- Real-time predictions
- Swagger UI testing

---

## Tools Used

- Python
- Pandas
- Scikit-learn
- FastAPI
- Uvicorn
- Joblib
- UV

---

## Run API

```bash
uvicorn app:app --reload
```
